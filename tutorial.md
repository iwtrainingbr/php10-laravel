
## Criar a estrutura do banco
- Definir o esquema do banco (nao precisa de codigo)
- Criar a migration (criar o arquivo via linha de comando)
- Criar o model
- Alterar a migration com os novos atributos/colunas
- Executo as migrations (php artisan migrate)

## Criar os endpoints
- Criar a classe de controller (php artisan make:controller)
- criar os metodos para cada endpoint
- definir as rotas que irao chamar esses metodos do controller
